Application Port Anomaly Detection Software

In this Project, I built and deployed an anomaly detection GUI application using python and tkinter to detect unauthorized open application ports and services running on servers in order to reduce cyber-attacks. This application works by consuming two NMap port scan reports. A baseline port scan report and a recent or new port scan report. It then compares the recent scan to the baseline scan and returns a detailed report on the changes in port status and services that have occurred in the new scan. These changes can then be investigated by the internal incidence response team and all malicious changes can be corrected before any harm is done. Since majority of cyber-attacks are perpetuated through maliciously opened ports, this solution reduces malicious attacks automatically by 70%.

Crypto-Currencies Correlation Analysis

In this project I undertook a correlation analysis study of ten (10) different crypto-currencies to examine the influence they exerted on each other's prices. I found out that that even though all nine (9) crypto-coins have good correlation matrixes with BTC and we can expect them to increase whenever BTC increases, some of the coins have even higher correlation matrixes with themselves than with BTC. This information is helpful in picking the coins to buy in order to maximize profits. and it also enables traders who like to diversify their crypto portfolio to invest in coins with very low correlation values so that if one coin dips the others won't.
